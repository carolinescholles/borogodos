# Borogodós

*é o plural de borogodó. O mesmo que: afetos, belezas, carinhos, charmes.

Quais são os meus afetos, belezas, carinhos e charmes? 
A ideia é escrever um post para cada afeto que eu tenho na minha vida.
Assuntos aleatórios.
